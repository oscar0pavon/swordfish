# Swordfish
Build software with 3D status progress like Swordfish movie

![idea](images/swordfish_movie.gif)  

# Dependencies

- vulkan drivers
- vulkan headers
- vulkan validation layers
- Xlib
- C compiler

# To test
    make
    sudo make install
Then in project directory for building  
    swordfish "command"  
Ex:  
    swordfish make -j8  
    swordfish ninja
